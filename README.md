# LinkHub

https://bento.me/kimmy-bento

Bento 스타일의 개인 링크 페이지 클론 프로젝트입니다.

## 미리보기

프로필과 다양한 소셜 링크를 벤토(Bento) 그리드 레이아웃으로 보여주는 정적 웹페이지입니다.

## 기능

-   프로필 섹션 (아바타, 이름, 바이오)
-   벤토 그리드 레이아웃 카드
-   Instagram/Behance 이미지 그리드
-   반응형 디자인
-   호버 애니메이션 효과

## 파일 구조

```
LinkHub/
├── index.html          # 메인 HTML 파일 (GitHub Pages용)
├── resources/          # 이미지 리소스 폴더
│   ├── profile.png     # 프로필 이미지
│   ├── instagram-*.jpeg # Instagram 이미지
│   ├── behance-*.jpeg  # Behance 이미지
│   ├── preview-*.jpeg  # 미리보기 이미지
│   └── icon-*.png      # 아이콘 이미지
├── sources/            # 원본 소스 파일 (참고용)
├── .gitignore
└── README.md
```

## 사용 방법

```bash
# 브라우저에서 열기
open index.html
```

## 기술 스택

-   HTML5
-   CSS3 (Flexbox, Grid)
-   Google Fonts (Noto Sans KR)
